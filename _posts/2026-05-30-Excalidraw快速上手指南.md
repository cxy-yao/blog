# Excalidraw 快速上手指南

> Obsidian 中的手绘风格绘图插件，适合画网络拓扑、流程图、架构图

## 基础操作

### 创建绘图
1. 新建 `.excalidraw` 文件：`Ctrl+P` → `Excalidraw: Create new drawing`
2. 或在笔记中嵌入：`Ctrl+P` → `Excalidraw: Embed new drawing`

### 常用快捷键

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 选择 | `V` 或 `1` | 选择/移动元素 |
| 矩形 | `R` 或 `2` | 画矩形 |
| 椭圆 | `O` 或 `3` | 画椭圆 |
| 菱形 | `D` 或 `4` | 画菱形（决策框） |
| 箭头 | `A` 或 `5` | 画箭头连接 |
| 直线 | `L` 或 `6` | 画直线 |
| 文字 | `T` 或 `7` | 添加文字 |
| 自由画 | `P` 或 `8` | 手绘线条 |
| 橡皮擦 | `E` 或 `9` | 删除元素 |
| 锁定工具 | 按住 `Shift` | 画正方形/圆形 |

### 编辑技巧
- **双击** 形状 → 直接编辑文字
- **拖动边缘** → 调整大小
- **按住 Alt 拖动** → 复制元素
- **Ctrl+Z / Ctrl+Y** → 撤销/重做
- **Ctrl+A** → 全选
- **Ctrl+D** → 复制选中元素
- **Ctrl+G** → 组合元素

## 画网络拓扑图

### 常用形状

| 设备类型 | 推荐形状 | 颜色建议 |
|----------|----------|----------|
| Spine 交换机 | 矩形（圆角） | 蓝色 `#a5d8ff` |
| Leaf 交换机 | 矩形（圆角） | 绿色 `#b2f2bb` |
| 服务器 | 矩形 | 灰色 `#e9ecef` |
| 防火墙 | 菱形 | 红色 `#ffc9c9` |
| 路由器 | 椭圆 | 紫色 `#d0bfff` |
| 云/外部 | 椭圆 | 橙色 `#ffd8a8` |

### 画 Spine-Leaf 拓扑示例

```
1. 画 2 个 Spine（蓝色圆角矩形）放在顶部
2. 画 4 个 Leaf（绿色圆角矩形）放在下面
3. 用箭头连接每个 Spine 到每个 Leaf
4. 在下方画服务器（灰色矩形）连接到 Leaf
5. 添加文字标注：设备名、接口、IP 地址
```

### 标注技巧
- **设备名**：双击形状，输入 `Spine-1` 或 `CE12800-1`
- **接口信息**：在箭头旁边添加文字 `Eth1/1 ←→ Eth1/48`
- **IP 地址**：用小字标注 `10.1.10.1/30`
- **VLAN/ASN**：用不同颜色背景标注

## 画流程图

### 标准符号
- **矩形** → 处理步骤
- **菱形** → 判断/决策
- **椭圆** → 开始/结束
- **箭头** → 流程方向

### 故障排查流程示例

```
开始（椭圆）
    ↓
检查接口状态（矩形）
    ↓
接口是否 Up？（菱形）
    ├─ 是 → 检查路由表
    └─ 否 → 检查物理连接
              ↓
          光模块是否正常？
              ├─ 是 → 检查配置
              └─ 否 → 更换光模块
```

## 高级技巧

### 1. 组合元素
选中多个元素 → `Ctrl+G` → 整体移动

### 2. 对齐元素
选中多个元素 → 工具栏选择对齐方式（左/中/右/上/中/下）

### 3. 锁定比例
按住 `Shift` 画图 → 保持正方形/圆形

### 4. 网格吸附
- 点击右上角菜单 → 开启网格
- 元素会自动对齐到网格线

### 5. 导出图片
- 右上角菜单 → Export image
- 支持 PNG、SVG 格式
- 可选择是否包含背景

### 6. 嵌入到笔记
```markdown
![[拓扑图.excalidraw]]
```
在普通 Markdown 笔记中引用绘图文件

## 快速开始

1. **打开 Excalidraw**：`Ctrl+P` → `Excalidraw: Create new drawing`
2. **画第一个矩形**：按 `R`，在画布上拖动
3. **添加文字**：双击矩形，输入文字
4. **画箭头**：按 `A`，从一个形状拖到另一个
5. **调整颜色**：选中元素，在工具栏选择颜色
6. **保存**：`Ctrl+S`

## 进阶功能

## 常见问题

### Q: 如何画直角箭头？
A: 画箭头时，按住 `Shift` 可以画出水平/垂直/45度的直线

### Q: 如何精确对齐？
A: 开启网格吸附，或使用对齐工具（选中多个元素后出现）

### Q: 如何复制样式？
A: 选中元素 → `Ctrl+Shift+C` 复制样式 → 选中目标元素 → `Ctrl+Shift+V` 粘贴样式

### Q: 如何导出高清图片？
A: 导出时选择 SVG 格式，或设置 PNG 导出比例为 2x 或 3x

### Q: 如何在绘图中插入代码？
A: 使用文字工具，输入代码内容，可以用等宽字体
