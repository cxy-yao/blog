## Giscus 评论系统集成测试

这是一篇测试文章，用于预览 **Giscus 评论系统** 的布局效果。

Giscus 是一个基于 GitHub Discussions 的评论系统，它具有以下特点：

- 🔒 **零广告、零追踪** — 完全基于 GitHub 平台
- 🎨 **支持自定义主题** — 可适配深色/浅色主题
- 📦 **无需数据库** — 所有评论数据存储在 GitHub Discussions 中
- 🌐 **支持 Markdown** — 评论支持 Markdown 格式和表情符号

### 安装步骤

要在你的 Jekyll 博客中集成 Giscus，你需要：

1. **启用 GitHub Discussions** — 在仓库 Settings 中开启
2. **安装 Giscus App** — 授权 Giscus 访问仓库
3. **获取配置 ID** — 在 giscus.app 页面配置后获取
4. **添加脚本** — 在文章页面添加 Giscus 客户端脚本
